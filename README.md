ИНСТРУКЦИЯ:

1.Распакуйте архив metrostroi_couple_fix_v2 в любое место

2.Из распакованного архива скопируйте папку metrostroi_couple_fix и вставте ее по пути: Диск и папка где у вас установлен стим:/steam:/steamapps:/common:/GarrysMod:/garrysmod:/addons ( ПРИМЕР ПУТИ: D:\Games\Steam\steamapps\common\GarrysMod\garrysmod\addons)

Дополнительно: Если у вас есть выделенный сервер , то вставьте папку из архива по пути: Диск и папка с SteamCMD:SteamCMD\steamapps\common\GarrysModDS\garrysmod\addons ( ПРИМЕР ПУТИ: D:\Games\SteamCMD\steamapps\common\GarrysModDS\garrysmod\addons)

Дальше запускайте гаррис мод и проблема должна исчезнуть
